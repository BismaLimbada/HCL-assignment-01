Core Competencies

Norman's Action Cycle Analysis
Can identify all seven stages: Goal, Intention, Specification, Execution, Perception, Interpretation, and Evaluation. Able to identify Gulf of Execution and Gulf of Evaluation in interface designs.
GOMS Modeling
Can decompose tasks into Goals, Operators, Methods, and Selection Rules. Able to model expert user procedures and predict approximate completion times.
KLM Time Prediction
Can identify KLM operators (K, P, H, M, R) and calculate execution time predictions. Understands model assumes error-free expert performance.
Safety-Critical System Evaluation
Demonstrated ability to apply multiple models together for comprehensive analysis. Can use GOMS for efficiency and Norman for error identification.

Case Study: Medical Infusion Pump

Applied all three cognitive models to analyze a safety-critical medical device, identifying:

  • Execution time predictions using KLM (2.4 seconds for rate entry)
  • Expert workflow modeling using GOMS
  • Error sources using Norman's Gulf of Evaluation
  • Design recommendations for both speed and safety

Standard Operator Set (Dix et al.)

The Keystroke-Level Model reduces complex nurse-machine interactions into seven primitive physical and mental units.

KLM Primitive Operators
Symbol Operator Name Description
K Keystroke Striking a key (e.g., dosage number).
B Button Press Pressing/releasing a mouse or joystick button.
P Pointing Moving a cursor to a target on screen.
H Homing Switching hands between controls (e.g., dial to screen).
D Drawing Tracing a path on a grid.
M Mental Prep Psychological pause for decision making.
R Response System wait time (e.g., machine calibration).

Computational Example: Reset Alarm

Scenario: A nurse must silence a "High Pressure" alarm and reset the flow rate.

Task_Total = M + P + K + P + M + K

1. Recognize Alarm (M)
2. Reach for Silence (P)
3. Press Silence (K)
4. Reach for Reset (P)
5. Verify Code (M)
6. Press Reset (K)

Analysis: By minimizing the number of H (Homing) and P (Pointing) operators, engineers can reduce the time-to-safety during emergencies.

Model Limitations

KLM assumes an Expert User with Zero Errors. It does not account for the stress or confusion a nurse might feel during a Code Blue, which is why Norman's Gulf of Evaluation is also required.

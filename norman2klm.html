Standard Operator Set (Dix et al.)

The Keystroke-Level Model reduces complex nurse-machine interactions into primitive physical and mental units.

KLM Primitive Operators
Symbol Operator Name Time Description
K Keystroke 0.20 sec Pressing a key or button.
P Pointing 1.10 sec Moving cursor to a target.
H Homing 0.40 sec Switching hands between controls.
M Mental Prep 1.35 sec Mental preparation/thinking time.
R Response Variable System wait time (e.g., display update).

Computational Example: Setting Rate

Scenario: Nurse sets infusion rate to 10 ml/hr using numeric keypad.

Task Sequence:

M - Mentally prepare to enter rate
K - Press Rate button
K - Press digit "1"
K - Press digit "0"
K - Press Enter
R - Wait for display update (0.2s)

Time Calculation:
M = 1.35s
K + K + K + K = 4 × 0.20s = 0.80s
R = 0.20s
────────────
Total Time = 2.35 seconds

Analysis: KLM predicts an expert nurse can set the rate in approximately 2.4 seconds with error-free performance.

Model Limitations

KLM assumes an Expert User with Zero Errors. It does not account for stress, confusion, or interpretation difficulties. This is why Norman's Gulf of Evaluation analysis is also required for medical devices.

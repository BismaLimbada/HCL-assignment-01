Project Constraints

This assignment follows the strict constraints of Lab 01 to demonstrate mastery of HTML foundations:

  • Semantic HTML: Use of <header>, <nav>, <main>, <section>, and <footer>.
  • Accessibility: Proper use of aria-current, alt text, and <label for>.
  • Minimal CSS: No layout or box-model properties (no margins, padding, or flexbox).

Exam Preparation (Part A: Common Mistakes)

Common HTML Mistakes vs. Correct Implementation
Error Type Incorrect Method Correct Semantic Method
Navigation Placing <a> directly inside <ul>. Nesting <a> inside <li> inside <ul>.
Hierarchy Skipping heading levels (e.g., H1 to H3). Sequential headings (H1, H2, H3).
Forms Input fields without connected labels. Using for attribute in label matching the input id.

Theoretical Knowledge

The analysis conducted in this project proves that while GOMS and KLM are excellent for measuring expert speed, Norman's Action Cycle is essential for identifying safety risks (The Gulf of Evaluation) in medical hardware like automatic syringes.
